ℹ️ OBS Setup

Browser Source Recommended

  1. Enter or generate a Room Code, then click Connect
  2. Copy the OBS URL that appears
  3. In OBS add a Browser Source
  4. Paste the OBS URL. Default size is 315 × 450
  5. Change OBS Overlay Scale in Settings to resize. The OBS URL auto-updates.
  6. The overlay background is transparent — no chroma key needed

Popup Window

Click Open OBS Popup to open a popup window. Capture it with Window Capture in OBS. Set custom background color in Settings → OBS Background Color.
